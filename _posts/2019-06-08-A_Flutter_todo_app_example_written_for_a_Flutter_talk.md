This project is part of a talk introducing Flutter to new developers.

Are you new to Flutter? Start [here](https://flutter.io/) !

- Listing items
- Adding items
- Removing items
- Editing items
- Data persistence

- Shared Preferences (Data Persistence)

- Integration tests (Automation)
